Задача
Каталог спецтехники, по которому удобно выбирать

Rippa продаёт мини-экскаваторы, мотоблоки и навесное оборудование через сеть дилеров по всей России. Сайт был устаревшим: плоский список техники без фильтров, характеристики в PDF, форма заявки только в подвале.

Дилеры жаловались, что клиентам проще позвонить напрямую, чем разобраться на сайте. Задача — сделать каталог, который помогает выбрать и сразу оставить заявку.

Решение
Что мы сделали
01
Каталог с фильтрами

Спроектировали умный каталог с фильтрацией по классу техники, мощности, весу и типу работ. Технические характеристики вынесли прямо в карточку — без единого PDF

02
Страницы товаров

Каждая модель получила свою страницу: галерея с видео, сравнение модификаций, таблица характеристик, применение и карта с ближайшим дилером

03
Форма заявки

Плавающая кнопка заявки на всех страницах каталога, выбор региона для привязки к дилеру, интеграция с CRM через API — заявка попадает к нужному дилеру автоматически
